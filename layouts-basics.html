Basic Layout

The layout system uses containers, rows, and columns to structure content. Helper classes like center, self-h-center, and self-v-center allow alignment of content horizontally, vertically, or both. The .container--fit class ensures the container adapts to the full width of its parent.

  • .container — main wrapper for layout sections is flex container with flex direction column
  • .container--fit — expands the container to fit the available width of 1200px
  • .row — flex container for columns
  • .column — individual flex item within a row
  • .center — horizontally centers content inside a column
  • .self-h-center — centers a single element horizontally within its parent
  • .self-v-center — centers a single element vertically within its parent

This layout system helps build responsive and flexible structures. Use rows to group columns, and use the centering classes to align content as needed.

Example markup:



<!-- Row with three centered columns -->

<div class="row">

  <div class="column center">

    <p>Column 1 content centered</p>

  </div>

  <div class="column center">

    <p>Column 2 content centered</p>

  </div>

  <div class="column center">

    <p>Column 3 content centered</p>

  </div>

</div>


<!-- Single item centered horizontally and vertically -->

<div class="self-h-center self-v-center">

  <p>This box is centered both horizontally and vertically.</p>

</div>
