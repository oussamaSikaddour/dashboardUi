Footer Layout

The .footer class defines the main footer container used to display essential links, brand identity, and social media icons. It’s flexible, responsive, and keeps all elements centered and visually balanced across different screen sizes.

  • .footer — root container for the footer. Uses a column layout with centered items and consistent padding.
  • .footer__container — defines the inner footer area with background color and horizontally distributed elements.
  • .footer__socials — list of social icons aligned in a row, each styled with circular borders and hover animations.
  • .footer__info — section displaying the logo and footer text (like copyrights).
  • .footer__info::before — creates a horizontal line separator above the info section.

The footer supports accessibility, responsive design, and elegant hover interactions for the social icons.

Example markup:


<footer class="footer">
  <div class="footer__container">
    <ul class="footer__socials">
      <li>
        <a href="#" aria-label="Facebook">
          <i class="fab fa-facebook-f"></i>
        </a>
      </li>
      <li>
        <a href="#" aria-label="Twitter">
          <i class="fab fa-twitter"></i>
        </a>
      </li>
      <li>
        <a href="#" aria-label="LinkedIn">
          <i class="fab fa-linkedin-in"></i>
        </a>
      </li>
    </ul>
  </div>

  <div class="footer__info">
    <img src="assets/img/Logo.png" alt="Company logo" class="logo" />
    <p>© 2025 DashboardUi. All rights reserved.</p>
  </div>
</footer>
