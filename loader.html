Loader

Loader

The Loader component provides a visual indication that a process is ongoing (e.g., loading data, submitting a form, or processing an action). It consists of two animated circles rotating in opposite directions to create a smooth loading motion.

  • .loader — the main loader container that defines the size and animation timing.
  • .loader__circle — represents one of the two animated circles composing the loader animation.
  • .loader.l — large version of the loader (used for page-level loading states).
  • .loader.xs — extra-small loader (commonly used inside buttons or small areas).
  • .loader__container — full-screen loader container that centers the loader and overlays the interface.
  • .loader__container.show — makes the full-screen loader visible with a blur and gradient background.

The loader supports multiple sizes and can be displayed inline or as a full-page overlay using .loader__container. It can also be paired with buttons to show loading states during actions.

Example markup:


<div class="loader xs">
  <div class="loader__circle"></div>
  <div class="loader__circle"></div>
</div>

<div class="loader">
  <div class="loader__circle"></div>
  <div class="loader__circle"></div>
</div>

<div class="loader l">
  <div class="loader__circle"></div>
  <div class="loader__circle"></div>
</div>

<div class="loader__container show">
  <div class="loader l">
    <div class="loader__circle"></div>
    <div class="loader__circle"></div>
  </div>
</div>

CSS:


.loader {

  position: relative;
  width: 4rem;
  height: 4rem;
  animation: loading 10s linear infinite;
  margin: auto;

}

.loader__circle {

  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  background: var(--clr-dark);
  border-width: 0.2rem;
  border-style: solid;
  border-color: var(--clr-light);
  border-radius: 50%;
  animation: animateLoadingCircle 3s linear infinite;

}

.loader__circle:nth-child(2) {

  background: var(--clr-light);
  border-color: var(--clr-dark);
  animation-delay: -1.5s;

}

.loader.l {

  width: 8rem;
  height: 8rem;

}

.loader.xs {

  width: 2.3rem;
  height: 2.3rem;

}

.loader__container {

  display: none;
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  justify-content: center;
  align-items: center;
  background: linear-gradient(to bottom right, #087EA4, rgb(78.6046511628, 205.523255814, 246.3953488372), var(--clr-dark));
  -webkit-backdrop-filter: blur(1rem);
  backdrop-filter: blur(1rem);
  z-index: -1000;
  opacity: 0;
  transition: all 0.5s;

}

.loader__container.show {

  display: flex;
  z-index: 1000;
  opacity: 1;
  
}
