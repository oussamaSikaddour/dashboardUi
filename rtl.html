RTL Layout Documentation

This document explains how the .rtl class modifies your website‚Äôs layout and alignment for right-to-left languages like Arabic or Hebrew. It describes what each CSS section does and why it‚Äôs needed.

üåç Base Direction

  • Applying html.rtl changes the global reading flow to right-to-left.
  • Code blocks remain left-to-right for better readability using: html.rtl code { direction: ltr; }
  • Certain UI elements like .combobox__group and .pagination__links also stay LTR to preserve visual logic.

üßæ Tables and Data Alignment

  • The main table uses direction: rtl; to reverse reading order.
  • Containers and nested divs alternate directions to maintain correct scrollbar and text flow behavior: .table__container { direction: ltr; } and .table__container > div { direction: rtl; }.
  • Table headers and first cells are anchored to the right using right: 0;.
  • Text inside <th> and <td> aligns to the right: text-align: right;.
  • Reset buttons and rounded corners swap sides using margin-right: auto; and border-bottom-left-radius.

üßÆ Forms and Inputs

  • Form containers use direction: rtl; so labels and fields match Arabic text flow.
  • Inner .forms elements stay ltr for consistent icon and number placement.
  • Error messages are aligned right with .input__error { text-align: right; }.
  • Labels and icons are flipped using: .input__label { right: 1rem; } and .input__span { left: 0; }.
  • Textarea labels follow the same positioning for visual harmony.

üß≠ Navigation and Menus

  • Navigation icons on mobile are flipped horizontally using: .nav--phone .nav__btn i { margin-right: auto; }.
  • Notification badges move to the left side with: .nav__btn .badge { left: -1rem; }.
  • The language menu opens from the left: .nav .lang__menu { left: 0; }.
  • Main navigation items align to the right, while submenus open toward the left: .nav__items--sub { left: 0; }.

ü¶∂ Footer and Section Titles

  • Footer text moves visually to the right using: .footer > p { margin-right: auto; }.
  • Decorative section lines align on the right side of titles using: .section h2::after { right: 0; }.

‚úÖ Summary

  • Add class="rtl" to your <html> tag to enable RTL mode.
  • The layout mirrors margins, alignments, and icon positions for proper RTL presentation.
  • Code blocks, pagination, and numeric fields remain LTR for usability.
