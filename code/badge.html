Button Badges

Badges are small indicators that can be attached to buttons or other elements to display counts, statuses, or notifications. Each badge uses the .badge class, with optional modifier classes such as .badge--warning or .badge--danger to indicate different states.

  • .badge — default badge style (used for success or primary indicators)
  • .badge--warning — warning state badge, typically styled in yellow/orange
  • .badge--danger — danger state badge, typically styled in red
  • .button — wrapper for the badge, showing it in context of a clickable element

Badges can be used to display numbers, counts, or alerts attached to a button or other elements. The style and color can be modified with the appropriate modifier classes.

Example markup:



  <div class="button">

    success badge <span class="badge">100</span>

  </div>


  <div class="button">

    warning badge <span class="badge badge--warning">50</span>

  </div>

  <div class="button">

    danger badge <span class="badge badge--danger">10</span>

  </div>
