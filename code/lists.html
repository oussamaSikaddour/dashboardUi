Usage of Lists

Lists are used to display items in an organized manner. There are two main types: unordered lists (<ul>) for items without a specific order, and ordered lists (<ol>) for sequences or steps. Each list item is contained within a <li> element.

  • <ul> — unordered list, used when item order does not matter
  • <ol> — ordered list, used for sequences or steps
  • <li> — list item, used inside both <ul> and <ol>
  • Lists can be nested for complex hierarchies
  • Accessible lists should use proper semantic markup and avoid using lists purely for layout

Lists are ideal for displaying structured content such as steps, instructions, or collections. Use ordered lists for procedures or rankings, and unordered lists for general items or bullet points.

Example markup:


<!-- Unordered List -->

<ul>

  <li>Apple</li>

  <li>Banana</li>

  <li>Cherry</li>

  <li>Mango</li>

</ul>

<!-- Ordered List -->

<ol>

  <li>Boil water</li>

  <li>Add tea leaves</li>

  <li>Pour into cup</li>

  <li>Add sugar or milk</li>

</ol>

CSS:



/* Remove list styles on ul, ol elements with a list role, which suggests default styling will be removed */

/* Remove default list styles for ul and ol */

ul,
ol {

  list-style: none; /* Removes bullets/numbers */
  font-size: 1.5rem;
  
}

/* Ensure lists with role="list" also have no style */

ul[role=list],
ol[role=list] {

  list-style: none;
  padding: 0;
  margin: 0;

}
