Buttons

The button system uses semantic HTML and a combination of utility and modifier classes. Every button begins with the base .button class, while visual variations are applied using modifiers like .button--primary, .button--danger or the optional .rounded class for circular icon buttons. Icons can be added using <i> elements from Font Awesome for visual clarity.

  • ".button " base class for sizing, spacing, font, and transitions
  • ".button--primary" changes the button color theme
  • ".rounded" creates circular icon-only buttons
  • "<i class='fa-solid â€¦'></i> " optional icon

Example markup:


<button class="button button--primary">
  Click me <i class="fa-solid fa-check"></i>
</button>
