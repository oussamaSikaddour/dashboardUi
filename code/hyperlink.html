Usage of Hyperlinks

Hyperlinks allow users to navigate to other pages or sections of a website. They are created using the <a> element with an href attribute specifying the destination URL. Hyperlinks can be placed within text, paragraphs, lists, or buttons.

  • <a> — the anchor element used to create a hyperlink
  • href — specifies the destination URL or page
  • Hyperlinks can be styled with CSS for color, underline, hover effects, or other visual cues
  • Accessible hyperlinks should include descriptive text that explains the link’s purpose

Hyperlinks are fundamental to web navigation. Always provide meaningful text so users and screen readers understand where the link leads.

Example markup:


<p>

  This is a test of the hyperlink 

  <a href="index.html">Home</a>

</p>

CSS:



/* Default styles for all anchor elements */

a:not([class]) {

  color: #087EA4;
  text-decoration: none;
  outline: none; /* Remove default focus outline for mouse users */
  /* Consolidate transitions to one line for conciseness */
  transition: color 0.3s, -webkit-text-decoration 0.3s;
  transition: color 0.3s, text-decoration 0.3s;
  transition: color 0.3s, text-decoration 0.3s, -webkit-text-decoration 0.3s;
  /* CRITICAL: Accessible style for keyboard users */

}
a:not([class]):hover, a:not([class]):focus {

  /* Use the darken function to ensure the hover color is consistent with the primary color */
  color: rgb(5.6279069767, 88.6395348837, 115.3720930233);
  text-decoration: underline;
  /* Redundant outline: none; is correctly omitted here */

  
}

a:not([class]):focus-visible {

  outline: 0.2rem solid #087EA4;
  outline-offset: 0.2rem;

}
