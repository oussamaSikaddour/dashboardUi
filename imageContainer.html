Images Gallery Component

The Documents Gallery component provides an accessible, scrollable view of document images such as ID cards, proof of address, or diplomas. Each item is represented by a clickable button containing an image and a label.

  • .imgs__container — the main wrapper that defines the gallery area with padding, rounded corners, and shadow.
  • .imgs — horizontal list container that scrolls if content overflows. It uses flex layout with spacing and alignment.
  • .img__button — interactive wrapper for each document. It includes the image, handles hover effects, and maintains consistent size and style.
  • .img — the image element itself, fitted proportionally within a fixed-size frame using object-fit: contain;.
  • .img__label — label placed below each image to identify the document.

This component is ideal for file upload previews, document galleries, or verification panels. Each button can be extended to trigger a modal or file viewer when clicked.

Example markup:



<div class="imgs__container">

  <h2>List of Image</h2>

  <div class="imgs" role="list" aria-label="List of Images">

    <button class="img__button" aria-label="View image two">

      <img

        class="img"

        src="assets/default/img-1.jpg"

        alt="Image One"

      />

    </button>

    <button class="img__button" aria-label="View image two">

      <img

        class="img"
        src="assets/default/img-2.jpg"
        alt="Image Two"

      />

    </button>

  </div>

</div>
