This CSS file defines the foundational styles, color palette, typography, animations, and accessibility utilities used throughout the application. It establishes global variables in :root, resets browser defaults, and introduces reusable animations.

üé® Root Variables

The :root section defines CSS custom properties (variables) for colors, typography, and common layout settings. These are used throughout the application to maintain consistent styling.

  • Color Palette: Includes variables for primary, secondary, success, danger, warning, info, accent, and neutral tones (both light and dark variations).
  • Background Colors: Variables for --clr-bg-light, --clr-bg, and --clr-bg-dark.
  • Gradients: Predefined gradient variables like --gradient-primary and --gradient-success.
  • Typography: Uses the Ubuntu font via Google Fonts and defines base font size, line height, and radius variables.
  • Layout and Shadows: Includes reusable values for spacing, borders, and shadow effects.
:root {
  --clr-primary: #087EA4;
  --clr-secondary: #6366f1;
  --clr-danger: #dc2626;
  --ff: "Ubuntu", sans-serif;
  --br: 0.8rem;
  --gradient-primary: linear-gradient(135deg, var(--clr-primary), #0ea5e9);
}

üåç Global Resets

The global reset normalizes layout behavior and improves consistency across browsers.

  • Removes default margins and paddings.
  • Improves scroll behavior with scroll-behavior: smooth.
  • Customizes ::selection colors for better contrast.

‚ú® Animations

Several keyframe animations are defined for motion effects, such as sliding, fading, scaling, and parallax. These are applied to interactive UI elements like modals, buttons, loaders, and banners.

  • slideInCentre ‚Äì slides content from below with opacity.
  • scroll ‚Äì creates a horizontal marquee scrolling effect.
  • scaleInCenter ‚Äì scales elements from center for pop effects.
  • slideInLeft / slideInRight ‚Äì directional slide-in animations.
  • fadeIn ‚Äì smooth fade-in transition.
  • animateButtonIcon ‚Äì shifts button icons horizontally on hover.
  • bgSlide ‚Äì simulates background parallax motion.
  • arrowSlide ‚Äì vertical up/down arrow motion.
  • slideFromBottom ‚Äì introduces elements from the bottom of the screen.
  • animateLoadingCircle ‚Äì loading circle scale animation.
  • loading ‚Äì infinite rotation for loaders or spinners.
