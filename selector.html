Select / Dropdown Inputs

The select component uses semantic HTML and BEM-structured utility classes to create accessible dropdowns with optional icons and validation messages. Every select begins with the base .select class, while supporting elements like .select__label, .select__item, and .select__error provide structure for options and feedback.

  • .select — base class for dropdown styling, spacing, and behavior
  • .select__group — wrapper that positions label, select box, and icons
  • .select__label — label describing the dropdown
  • .select__item — individual option or item inside the select
  • .item__error — validation message shown on error

Options are wrapped inside .select__group for layout consistency. Each item is marked with .select__item for styling and accessibility. Validation messages appear beneath the select using .item__error.

Example markup:


<div class="select__input">
  <div class="select__group">
    <label class="select__label" for="selectOne">Choose an Option:</label>
  <div  class="select" >
    <select id="selectOne"  required>
      <option class="select__item" value="" disabled selected>--Select--</option>
      <option class="select__item" value="option 1">Option One</option>
      <option class="select__item" value="option 2">Option Two</option>
    </select>
  </div>
  </div>
  <div class="item__error">Please select an Option.</div>
</div>
