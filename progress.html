Usage of Progress Bars

The <progress> element represents the completion progress of a task. It can be used to show file uploads, downloads, or any operation with a measurable progress value. The element uses the value attribute for the current progress and the max attribute for the maximum value.

  • <progress> — native HTML element for showing progress
  • value — current progress value
  • max — maximum value indicating completion
  • Accessible progress bars should always include the value and max attributes
  • Text labels can be added nearby to indicate the progress to users

Progress bars are useful for providing feedback to users during ongoing operations. They can be styled with CSS to fit the design system, or enhanced with animations.

Example markup:


    
<p>Downloading file...</p>

<progress value="70" max="100"></progress>

Css:



progress[value] {

  appearance: none; /* remove default styling across browsers */
  -webkit-appearance: none;
  -moz-appearance: none;
  width: 20rem;
  height: 2rem;
  margin: 0 1rem;
  border: 0.1rem solid #CED0D4;
  border-radius: var(--br);
  background: #E9EAF4;
  box-shadow: var(--div-shadow);
  /* WebKit browsers (Chrome, Safari) */
  /* Firefox */

}

progress[value]::-webkit-progress-bar {

  border-radius: var(--br);
  background: #E9EAF4;

}

progress[value]::-webkit-progress-value {

  border-radius: var(--br);
  background: #087EA4;
  -webkit-transition: width 0.3s ease;
  transition: width 0.3s ease;

}

progress[value]::-moz-progress-bar {

  border-radius: var(--br);
  background: #087EA4;
  -moz-transition: width 0.3s ease;
  transition: width 0.3s ease;

}
