Form Image Upload

The image upload component allows users to select images using a button that visually represents the uploaded file. The main wrapper is .file__input__item, and the clickable button with placeholder image is contained in .image__input__group. For profile images, the .profile-image class modifies the layout or styling to suit a user profile context. Validation messages appear below the group using .input__error.

  • .file__input__item — main wrapper for the image upload component
  • .image__input__group — holds the button and file input
  • .image__input__placeholder — default or previously uploaded image
  • .image__input__button — camera icon indicating clickable upload
  • button — triggers file selection
  • input[type="file"] — native file input element
  • .profile-image — optional modifier class for profile image layout
  • .input__error — validation message shown on error

Users can click on the button containing the placeholder image to open the file selector. The profile image variant is styled differently to suit avatar or user picture usage. Validation messages can be toggled under the component when needed.

The JavaScript here works the same way as in the File Input example.

Regular Image Upload markup:


<!-- Regular Image Upload -->

<div class="file__input__item" role="input">

  <div class="image__input__group">

    <button class="button">

      <img src="./assets/img/picture.png" alt="defaultImg" class="image__input__placeholder" />

      <img src="./assets/img/camera.png" class="image__input__button" />

    </button>

    <input type="file" name="myfile" />

  </div>

  <div class="input__error">

    ⚠ This error text will show only when there is an error

  </div>

</div>

<!-- Profile Image Upload Variant -->
