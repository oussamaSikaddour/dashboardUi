Section Layout

The .section class defines a visual content block used to separate and structure different parts of the interface. It provides consistent padding, background color, border styling, and rounded corners for better readability.

  • .section — wrapper for a distinct part of the layout. It includes padding, a light background, and a bottom border for visual separation.
  • h1 / h2 — section headings with consistent margins, font weights, and an underline accent for h2.
  • p — paragraphs are styled for optimal line height and readability (up to 95 characters per line).
  • ul, li — lists have proper spacing and indentation for clarity.
  • table — fits naturally within the section layout, ensuring the table width adapts to its content.

Sections can be stacked vertically in a page to organize documentation or dashboards. The last section automatically removes the bottom border for a clean finish.
